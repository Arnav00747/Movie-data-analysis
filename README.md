# 🎬 Movie Data Analysis

## 📌 Project Overview
This project analyzes a large movie dataset (TMDB + IMDB) to extract insights about genres, revenue, ratings, and trends.

---

## 📊 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Genre-wise Analysis
- Revenue vs Rating Analysis
- Interactive Dashboard (HTML)

---

## 📁 Files in this Repository
- `MOVIE_ANALYSIS.ipynb` → Data analysis notebook
- `index.html` → Dashboard visualization

---

## 📊 Insights
- Adventure and Animation genres generate higher profits
- Genre plays a key role in movie success
- Revenue trends vary significantly across years

---

## ⚠️ Dataset
Due to large file size, dataset is not uploaded on GitHub.

👉 Download Dataset from:
https://www.kaggle.com/datasets/ggtejas/tmdb-imdb-merged-movies-dataset

## 🛠️ Tech Stack
- Python (Pandas, Matplotlib)
- Jupyter Notebook
- HTML, CSS

---

## 🚀 How to Run
1. Open `MOVIE_ANALYSIS.ipynb` in Jupyter Notebook
2. Run all cells
3. Open `index.html` in browser for dashboard
