# 🎬 Movie Data Analysis

## 📌 Project Overview
This project analyzes a large movie dataset (TMDB + IMDB) to extract insights about genres, revenue, ratings, and trends.

---

## 📊 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Genre-wise Analysis
- Revenue vs Rating Analysis
- Interactive Dashboard (HTML)

---

## 📁 Files in this Repository
- `MOVIE_ANALYSIS.ipynb` → Data analysis notebook
- `index.html` → Dashboard visualization

---

## 📊 Insights
- Adventure and Animation genres generate higher profits
- Genre plays a key role in movie success
- Revenue trends vary significantly across years

---
